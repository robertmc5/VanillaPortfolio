Movie Hunt

Search for movies you might want to see. This application is written with vanilla JavaScript. It fetches from the TMDB API.

I draw from multiple endpoints revealed in the specs of the API to display the movie information I would want to see. I group twenty movies at a time so a bunch can be scrolled over quickly. A trailer can be viewed by means of an iFrame from YouTube.

The movie data varies so error handling, data validation and messaging in the DOM is utilized when needed. Of course it's fully responsive.
