Movie Hunt

Search for movies you might want to see. This application is written with vanilla JavaScript. It fetches from the TMDB API.

I draw from multiple endpoints revealed in the specs of the API to display the movie information I would want to see. I group twenty movies at a time so a bunch can be scrolled over quickly. A trailer can be viewed by means of an iFrame from YouTube.

The movie data varies so error handling, data validation and messaging in the DOM is utilized when needed. Of course it's fully responsive.

Wordsmith

Get word suggestions based on some criteria. This responsive app makes requests to the Datamuse API. It uses JavaScript to combine event listeners and visually active panels so the user can readily choose their desired capability.

The HTML Markup utilizes data-* attributes to relay the query string for the chosen API endpoint resources. The responses are managed to display a clean, appropriately sized list or instructive message.

Emmyville

This is a personal site just for kicks. I plan to keep adding games or visuals to it as I have time. The theme is our cat, Emmy, and I'm using JavaScript to create game logic, interactivity, and DOM dynamics for any sized device.

There are three games currently available. Concentration uses a timer and an animated flipped card effect. Name Guesser has multiple inputs and responses and transforms words visually on the screen. I discovered a matchMedia method that was new to me that adapts the behavior based on viewport size. The main claim to fame of Magic 8-ball is bad cat puns.
